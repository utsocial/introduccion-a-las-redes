Figure 1 shows a table with all the possible subnets for 10.x.0.0/16. The first column is the subnet address. Although there are 256 possibilities, only the first seven and the last subnet are shown. The second column shows the host range for each subnet. Each subnet can support 65,534 hosts. The third column shows the broadcast address for each subnet. Figure 2 shows a table with all the possible subnets for 10.x.x.0/24. Although there are 65,536 possibilities, only nine are shown. The second column shows the host range for each subnet. Each subnet can support 254 hosts. The third column shows the broadcast address for each subnet.
